## 📖 Description du projet

M&N Cosmetics est un site web e-commerce spécialisé dans les produits cosmétiques tunisiens. Le site permet aux utilisateurs de :
- S'inscrire avec validation robuste
- Consulter le catalogue de produits
- Gérer un panier d'achat
- Persister leurs données avec localStorage

**Technologies utilisées :**
- HTML5
- CSS3 (Responsive Design)
- JavaScript ES6+
- Git & GitHub

---

## ✨ Fonctionnalités

### 1. Page d'accueil (page1.html)
- Présentation du site
- Navigation principale
- Design attrayant avec dégradé

### 2. Page d'inscription (page2.html)
- Formulaire d'inscription complet
- Validation email robuste (regex)
- Validation password forte (8+ chars, majuscule, chiffre, spécial)
- Confirmation email
- Messages d'erreur clairs
- Focus states élégants

### 3. Page produits (page3.html)
- Catalogue de produits
- Images optimisées (WebP + lazy loading)
- Fiches produits détaillées
- Avis clients visibles

### 4. Page panier (page2.html - section)
- Gestion dynamique du panier
- localStorage pour persistance
- Toast notifications
- Suppression d'articles possible

### 5. Design Responsive
- Mobile first approach
- Breakpoints: 480px, 768px, 1024px
- Testé sur tous les appareils
